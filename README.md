# Salaty App (Version 2)

**Salaty** is a lightweight, clean, and completely **offline** (after first launch) prayer times and azkar application.

## Main Features

- Prayer times with countdown to the next prayer  
- Current Hijri & Gregorian date  
- Qibla direction (for devices with compass sensor)  
- Electronic Tasbih / Dhikr counter (custom dhikr + target count)  
- Post-prayer Azkar (remembrance after Salah)  
- Days remaining until major Hijri events  
  (Ramadan, Eid al-Fitr, Eid al-Adha)  
- **No ads** at all  
- **Fully offline** after loading fonts once (internet recommended only on first run)  
- Flexible settings:  
  - Change prayer time calculation method  
  - Select jurisprudential school (Madhhab)

## Why Salaty?

Simple. Fast. No distractions. Focused on helping you pray and remember Allah with ease.

## Available Customizations

- Multiple prayer time calculation methods (Muslim World League, Umm al-Qura, Egyptian, Shafi'i, Hanafi, ...)  
- Madhhab selection  
- Customizable dhikr counter
